# Handwritten_digit_recognition
# Handwritten Digit Recognition using Neural Network

## Description
This project implements a handwritten digit recognition system using a
feedforward neural network trained on the MNIST dataset.

## Dataset
- MNIST handwritten digits (28x28 grayscale images)

## Model
- Flatten layer
- Dense (128, ReLU)
- Dense (128, ReLU)
- Dense (10, Softmax)

## Tools Used
- Python
- TensorFlow / Keras
- OpenCV
- Google Colab

## Custom Image Testing
Custom 28x28 grayscale images are preprocessed and predicted using the trained model.
